# Cache local de espectros LAMOST DR9

Esta carpeta almacena automaticamente espectros FITS descargados desde LAMOST DR9.

- Patron de nombre: `spec_{obsid}.fits`
- Uso: cache local para evitar descargas repetidas durante el analisis.
- Versionado: esta carpeta no debe subirse al repositorio para evitar archivos binarios grandes.

Acceso manual a LAMOST DR9:
http://www.lamost.org/dr9/
